# AI 3D Playground

Generate and interact with 3D models using AI prompts.

## Demo

Type any object (robot, astronaut, horse) and explore it in interactive 3D.

## Tech Stack

Frontend:
- React
- Three.js
- React Three Fiber
- TailwindCSS

Backend:
- Node.js
- Express
- Sketchfab API

## Features

- Text → 3D model
- Interactive 3D viewer
- Model caching
- Loading progress UI

## Setup

### Backend

cd backend  
npm install  
node server.js  

### Frontend

cd frontend  
npm install  
npm run dev
